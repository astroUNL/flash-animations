Compiling these FLAs requires pointing to external code that is in the /codebase/edu/unl/astro folder of this repository.

Process:
- Open the FLA file.
- Go to File -> Publish Settings.
- Go to the "Flash" tab.
- Click the "Settings" button next to the Script drop-down.
- Go to the "Source path" tab.
- Add the following: "../../codebase"
